- ESLint

- Prettier

* Google Cloud Storage

# セットアップ

```
touch .env.local

# 依存パッケージのインストール
yarn

# 開発環境の起動
yarn dev
```

### バックエンドを立ち上げる()

https://github.com/tokio-k/group-manager-backend

### .env.local の内容

```
PROJECT_ID='プロジェクトID'
GOOGLE_APPLICATION_CREDENTIALS='認証情報のjsonファイルのパス'
```
